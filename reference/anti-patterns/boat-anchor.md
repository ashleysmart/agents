# Boat Anchor

A component, library, or abstraction kept in the codebase because it might be useful later, even though it serves no current purpose.

**Applies to**: Software Design

---

**What it is**: A component, library, or abstraction kept in the codebase
because it might be useful later, even though it serves no current purpose.

**Why not**: Dead weight. It increases build times, cognitive load, and
dependency surface. Delete it; version control preserves it if it is ever
genuinely needed.
