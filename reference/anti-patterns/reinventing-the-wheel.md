# Reinventing the Wheel

Implementing something that already exists as a well-tested, maintained library or standard library module.

**Applies to**: Software Design

---

**What it is**: Implementing something that already exists as a
well-tested, maintained library or standard library module.

**Why not**: Bespoke implementations carry bugs, maintenance burden, and
lack the battle-testing of established solutions. Every line of custom
code is a liability.
